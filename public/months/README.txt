Place your month hero images in this folder with these exact filenames:

january.jpg
february.jpg
march.jpg
april.jpg
may.jpg
june.jpg
july.jpg
august.jpg
september.jpg
october.jpg
november.jpg
december.jpg

Notes:
- You can use .jpg, .jpeg, .png, .webp, or .avif, but keep filenames in Calendar.tsx in sync.
- If a file is missing, the app will automatically fall back to a generated hero image for that month.
